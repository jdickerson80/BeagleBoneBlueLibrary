Detailed Description

The ServoControl class controls a servo motor. For convenience, this class adds API to control the servo in degrees instead of by pulse width or duty cycle. This allows easier control of the servo.

Note
This class can be controlled with pulse width and duty cycle if desired. The extra methods are just here for convenience.
