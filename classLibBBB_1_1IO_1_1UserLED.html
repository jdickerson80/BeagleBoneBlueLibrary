Detailed Description

The UserLED class represents a user controlled LED. This class allows the LED to blink in a threaded manner. Additionally, the LED can be controlled in a "manual" way.

Todo:
This class should inherit from IOBase, but Linux's LED control is significantly different than its GPIO control. However, IOBase can be refactored to accomodate the differences.
