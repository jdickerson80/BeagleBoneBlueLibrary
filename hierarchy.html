 CCore::IO::InputGPIO pin that is set up as an input. This class allows the user to add a callback function to either the rising, falling, or both edges of the input. This means that the function will automatically be called when its respective edge event happens. There are two different ways to use the edge callback: 1) waitForEdge() - where it blocks until the edge event happens. 2) waitForEdgeThreaded - which does the same thing, without blocking. If edge events are undesired, there are standard methods to just query the state of the input
