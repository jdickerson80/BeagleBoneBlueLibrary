Detailed Description

The BluetoothManager class manages the Bluetooth connection. It puts the socket into listen mode to connect to the remote device, and creates a read and write thread to handle sending and receiving data across the Bluetooth connection. This class also has a mutex and conditional variable to handle the race condition of the send buffer.
