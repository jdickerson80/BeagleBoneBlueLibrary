Detailed Description

The ManagerPRUs class manages BOTH PRUs for the application. This ensures that NO other part of the application can control the PRUs. In a basic sense, this class forces access to the pointer to PRU memory to come through it as a way of protecting a VERY important pointer. This forces the pointer's value to a constant that nothing but this class can modify.

uint32_t *const LibBBB::ManagerPRUs::sharedMemoryPointer ( )

sharedMemoryPointer gets the pointer to the shared memory pointer. This class has NO error checking method to ensure the pointer is used correctly. Most times the pointer is used as an array, so there is no error checking to make sure the app does not make a segmentation fault.

Note
Both getter methods do NOT allow the pointer to be changed to ensure it does not get moved to a position that will cause a segmentation fault.
Returns
pointer to the shared PRU memory
