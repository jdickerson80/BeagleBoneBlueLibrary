Detailed Description

The Output class represents a GPIO pin that is set up as an output. This class has the ability to toggle outputs for a specific number of times in a threaded or non-threaded manner. Additionally, there are methods to "manually" control the output.

Note
To construct an output, just create the setup struct, and pass the struct into the output's constructor. The setup struct, however, does not need to be kept in scope after it is passed to the constructor. This means that one could create the setup struct in the constructor of the output.
